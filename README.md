# Solar AI Platform 🌱☀️

A comprehensive web application designed to empower solar energy solutions using Artificial Intelligence. With an interactive and dynamically animated user interface, this application aims to monitor, predict, and maintain solar panel efficiency.

## 🚀 Features

- **Energy Prediction**: Upload historical solar data to accurately predict future energy output using AI models.
- **Fault Detection**: Detect visual or structural faults in solar panels via image uploads and AI analysis.
- **Image Classification**: Advanced algorithms to classify different solar hardware components.
- **Interactive UI**: Fluid animations powered by `framer-motion` and `aos` (Animate On Scroll) for a premium user experience.

## 📦 Tech Stack

- **React.js** (Create React App)
- **Framer Motion** & **AOS** for dynamic animations
- **Chart.js** for data visualization
- **SweetAlert2** for interactive notifications
- **Swiper** for modern sliders and carousels
- **React Router Dom** for seamless navigation
- **Vanilla CSS** with a custom design system for high-performance styling.

## 🛠️ Installation & Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/solar-ai.git
   ```
2. Navigate to the project directory:
   ```bash
   cd solar-ai
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 🌍 Deployment on Vercel

This application is fully optimized for continuous deployment on [Vercel](https://vercel.com/):
1. Push your code to your GitHub repository.
2. Go to your Vercel Dashboard and click **Add New Project**.
3. Import this GitHub repository.
4. Framework Preset will automatically be detected as **Create React App**.
5. Click **Deploy**. Vercel will install dependencies, build the app, and provide a live URL!
